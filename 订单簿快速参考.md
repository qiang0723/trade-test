## 📈 使用技巧

### 1️⃣ **快速判断买卖压力**

```
看背景条长度：
  长条 = 大量挂单 = 压力大/支撑强
  短条 = 少量挂单 = 压力小/支撑弱
```

### 2️⃣ **评估大单冲击**

```
看累计列：
  买入500单位 → 找卖单累计≥500的行
  → 估算平均成交价
```

### 3️⃣ **识别支撑/阻力**

```
找背景条最长的行：
  卖单区 → 阻力位
  买单区 → 支撑位
```
