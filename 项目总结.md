# BTC行情数据获取工具 - 项目总结

## 📦 项目概述

本项目提供了一套完整的加密货币（特别是BTC）行情数据获取工具，支持实时数据查询、数据导出、多币种对比等功能。

## 🎯 已完成功能

### 1. 核心功能
- ✅ 实时价格查询
- ✅ 24小时行情统计
- ✅ 订单深度（买卖盘）
- ✅ 最近成交记录
- ✅ K线数据（多种时间间隔）
- ✅ 数据导出（JSON + CSV）
- ✅ 多币种对比

### 2. 脚本文件

| 文件名 | 功能 | 适用场景 |
|--------|------|----------|
| `btc_market_simple.py` | 快速查看行情 | 终端直接显示，适合快速查看 |
| `btc_market_data.py` | 交互式查询 | 灵活选择数据，可自定义参数 |
| `btc_market_export.py` | 数据导出 | 保存为文件，适合数据分析 |
| `btc_market_multi_symbols.py` | 多币种对比 | 同时查看多个币种行情 |

### 3. 数据类型

#### 实时数据
- 当前价格
- 24小时最高/最低价
- 24小时涨跌幅
- 24小时成交量/成交额

#### 深度数据
- 买单深度（Bid）
- 卖单深度（Ask）
- 价格档位
- 挂单数量

#### 成交数据
- 成交时间
- 成交价格
- 成交数量
- 成交方向（买入/卖出）

#### K线数据
- 开盘价/收盘价
- 最高价/最低价
- 成交量/成交额
- 支持多种时间间隔（1m, 5m, 15m, 1h, 4h, 1d, 1w, 1M）

## 📊 使用示例

### 示例1：快速查看BTC行情

```bash
source venv/bin/activate
python3 btc_market_simple.py
```

**输出：**
```
💰 BTC 当前价格: $95,224.99 USDT
📊 24小时涨跌幅: 📉 -0.47%
📈 24小时最高: $95,809.08
📉 24小时最低: $94,293.46
```

### 示例2：导出数据用于分析

```bash
source venv/bin/activate
python3 btc_market_export.py
```

**生成文件：**
- `btc_market_20260117_174323.json` - 完整JSON数据
- `btc_ticker_24h_20260117_174323.csv` - 24小时统计
- `btc_klines_1h_20260117_174323.csv` - 1小时K线
- `btc_klines_1d_20260117_174323.csv` - 日线K线
- 等等...

### 示例3：多币种对比

```bash
source venv/bin/activate
python3 btc_market_multi_symbols.py
```

**输出：**
```
🏆 24小时涨幅榜 TOP 3:
  1. LTC        +3.62%  ($74.92)
  2. ETC        +2.78%  ($12.92)
  3. ATOM       +2.57%  ($2.55)

📉 24小时跌幅榜 TOP 3:
  1. DOGE       -1.43%  ($0.14)
  2. AVAX       -0.87%  ($13.67)
  3. BTC        -0.41%  ($95,266.72)
```

## 🔧 技术栈

- **语言**: Python 3.12
- **主要库**:
  - `python-binance` - 币安API客户端
  - `pandas` - 数据处理和分析
  - `requests` - HTTP请求
- **数据源**: 币安交易所 (Binance) 公开API

## ⚠️ 注意事项

### 1. API限制
- 币安API有请求频率限制
- 建议每次请求间隔至少0.2秒
- 避免短时间内大量请求

### 2. 网络要求
- 需要能够访问 api.binance.com
- 中国大陆用户可能需要使用代理

### 3. 数据准确性
- 数据来源于币安交易所
- 可能存在轻微延迟（<1秒）
- 仅供参考，不构成投资建议

### 4. 法律合规
- 遵守当地法律法规
- 注意加密货币交易的合法性
- 本工具仅用于学习和研究
