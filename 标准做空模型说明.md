# 标准做空模型评分说明

## 概述

标准做空模型与做多模型对称，通过5个核心条件对市场的做空机会进行量化评分，满分10分。

## 评分维度（满分10分）

### 1. 结构向下（或刚破位）- 0-2分

| 评分 | 条件 | 说明 |
|------|------|------|
| 2.0 | 24h跌幅>3% 且 6h延续下跌 | 结构明确向下，趋势清晰 |
| 1.5 | 6h跌幅>2% | 刚破位，破位初期 |
| 1.0 | -3% < 24h跌幅 ≤ 0 且 6h下跌 | 结构偏空，价格缓慢向下 |
| 0 | 不符合条件 | 无明确下跌结构 |

**说明**：
- 确认下跌趋势是做空的基础
- 刚破位的初期通常是较好的做空入场点
- 注意区分缓慢下跌和急速下跌

### 2. 下跌放量 / 反弹缩量 - 0-2分

| 评分 | 条件 | 说明 |
|------|------|------|
| 2.0 | 下跌时：成交量放大>15% | 下跌得到充分确认，抛压强 |
| 1.5 | 反弹时：成交量萎缩>10% | 反弹无力，卖压占主导 |
| 1.0 | 量能一般 | 成交量小幅变化 |
| 0 | 下跌缩量或反弹放量 | 量价背离，不支持做空 |

**说明**：
- 理想情况：下跌放量，反弹缩量
- 量价配合是趋势持续的关键
- 下跌缩量可能是洗盘，需谨慎

### 3. OI堆积或上涨配合下跌 - 0-2分

| 评分 | 条件 | 说明 |
|------|------|------|
| 2.0 | OI在8-15%区间 | 持仓堆积，风险积累 |
| 2.0 | 价格下跌>1% 且 OI上升>2% | 空头增仓，价格下行 |
| 1.0 | OI温和增长（2-8%） | 持仓小幅增加 |
| 0 | OI下降或极端波动 | 持仓减少或市场不稳定 |

**说明**：
- OI堆积（8-15%）通常预示风险积累，容易诱发回调
- 价格下跌时OI上升，说明空头正在主导
- OI极端波动（>15%）视为NO_TRADE信号

### 4. 资金费率过热或极度过热 - 0-1.5分

| 评分 | 条件 | 说明 |
|------|------|------|
| 1.5 | 资金费率 > 0.15% | 多头极度过热，空头获利 |
| 1.0 | 0.1% < 资金费率 ≤ 0.15% | 资金费率过热 |
| 0.5 | 0.08% < 资金费率 ≤ 0.1% | 资金费率偏高 |
| 0 | 资金费率 ≤ 0.08% | 资金费率正常或偏低 |

**说明**：
- 资金费率越高，多头持仓成本越大
- 极度过热（>0.15%）时，多头可能被迫平仓
- 资金费率极端（>0.2%）时，视为NO_TRADE信号

### 5. 主动卖单占优 - 0-2分

| 评分 | 条件 | 说明 |
|------|------|------|
| 2.0 | 卖出比例 53-65% | 卖单略占优，理想范围 |
| 1.0 | 卖出比例 65-70% | 卖单占优（偏强） |
| 0.5 | 卖出比例 > 70% | 卖单过强，杀跌风险 |
| 0.5 | 卖出比例 45-53% | 买卖均衡 |
| 0 | 卖出比例 < 45% | 买单占优，不适合做空 |

**说明**：
- 理想范围（53-65%）：卖盘略占优但不极端
- 卖单过强（>70%）可能是恐慌性杀跌，容易反弹
- 买卖均衡时，市场方向不明确

## 评分等级与信号

| 评分范围 | 信号强度 | 操作建议 |
|----------|----------|----------|
| ≥8分 或 完美符合 | 强烈做空 | 顺势做空，设置合理止损 |
| 6-7.9分 | 偏空 | 可考虑做空，控制仓位 |
| 4-5.9分 | 观望 | 信号不明确，建议观望 |
| <4分 | 不建议做空 | 不符合做空条件 |

## 完美做空信号（Perfect Short）

当以下5个条件同时满足时，判定为"完美做空信号"：

1. ✓ 结构向下（得分>0）
2. ✓ 下跌放量或反弹缩量（得分>0）
3. ✓ OI堆积或配合下跌（得分>0）
4. ✓ 资金费率过热（得分>0）
5. ✓ 卖单占优（得分>0）

## 风险提示

### ⚠️ NO_TRADE 条件（优先判断）

即使做空评分很高，如果出现以下极端情况，也应该判定为NO_TRADE：

1. **资金费率极端**：绝对值 > 0.2%
2. **OI极端变化**：变化超过±15%
3. **集中平仓**：OI大降（<-8%）+ 价格大幅波动（>5%）

### 止损建议

- **强烈做空（≥8分）**：止损设置在最近高点上方1-2%
- **偏空（6-7.9分）**：止损设置在最近高点上方2-3%
- **观望（<6分）**：不建议入场

### 仓位管理

- **强烈做空**：可考虑标准仓位（如账户20-30%）
- **偏空**：建议小仓位试探（如账户10-15%）
- **观望**：0仓位

## 使用场景

### 适用场景

1. **趋势明确的空头市场**：结构向下，量价配合
2. **多头过热后的回调**：资金费率过热，OI堆积
3. **破位后的追空**：刚破位，下跌放量

### 不适用场景

1. **震荡市**：无明确趋势，买卖均衡
2. **极端恐慌**：杀跌过猛，容易反弹
3. **底部区域**：价格已大幅下跌，风险收益比不佳

## 与做多模型的配合使用

### 同时参考两个模型

- **做多评分 vs 做空评分**：
  - 做空评分远高于做多评分（差距>3分）→ 空头信号强
  - 做多评分远高于做空评分（差距>3分）→ 多头信号强
  - 两者接近（差距<3分）→ 信号不明确，观望

### 三态交易信号

系统会综合做多模型、做空模型、极端条件，输出最终的三态信号：

- **LONG**：做多评分≥6 且 做多评分>做空评分+2
- **SHORT**：做空评分≥6 且 做空评分>做多评分+2
- **NO_TRADE**：出现极端条件 或 两个模型评分都不明确

## 实战案例

### 案例1：完美做空信号

```
24h跌幅：-4.2%，6h跌幅：-2.1% → 结构向下 ✓ (2分)
成交量变化：+18% → 下跌放量 ✓ (2分)
OI变化：+10.5% → OI堆积 ✓ (2分)
资金费率：+0.12% → 资金费率过热 ✓ (1分)
1h卖出比例：58% → 卖单略占优 ✓ (2分)

做空评分：9.0分 → 强烈做空
```

### 案例2：反弹缩量做空

```
24h跌幅：-1.5%，6h跌幅：+0.8% → 反弹 (1分)
成交量变化：-12% → 反弹缩量 ✓ (1.5分)
OI变化：+8.2% → OI堆积 ✓ (2分)
资金费率：+0.11% → 资金费率过热 ✓ (1分)
1h卖出比例：54% → 卖单略占优 ✓ (2分)

做空评分：7.5分 → 偏空
```

### 案例3：不建议做空

```
24h跌幅：-0.5%，6h跌幅：+1.2% → 反弹 (0分)
成交量变化：+5% → 量能一般 (0.5分)
OI变化：-2% → OI下降 (0分)
资金费率：+0.05% → 资金费率正常 (0分)
1h卖出比例：48% → 买卖均衡 (0.5分)

做空评分：1.0分 → 不建议做空
```
